# Epic Next.js 15 Tutorial: Learn Next.js by building a real-life project

This repository contains the **frontend** and **backend** implementations for a project based on Strapi's tutorial on building a real-world application using Next.js 14, Tailwind CSS, and Strapi. The course focuses on modern web development principles, including server-side rendering (SSR), dynamic routing, and API integration. It emphasizes practical techniques for building a professional-grade application while leveraging Tailwind's styling and Strapi's headless CMS capabilities.

The course covers the following lessons:
1. Getting Started with project
2. Building Out The Hero Section
3. Building Out The Features Section, TopNavigation and Footer
4. Building Out the Register and Sign In Page
5. Building out the Dashboard page
6. Get Video Transcript with OpenAI Function
7. Strapi CRUD permissions
8. Search & pagination

For the complete tutorial, visit the [Strapi](https://strapi.io/blog/epic-next-js-14-tutorial-learn-next-js-by-building-a-real-life-project-part-1-2?utm_campaign=DevRel%20Content&utm_source=Strapi%20Website&utm_content=epic-next-course-post-1) blog post.
